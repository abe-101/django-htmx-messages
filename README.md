Django+HTMX integration with the messages framework

## Installation

Install this via pip (or your favourite package manager):

`pip install django-htmx-messages`

Add the app to your `INSTALLED_APPS`:

```python
INSTALLED_APPS = [
    # ...
    "django_htmx_messages",
]
```
